## 1. 관심 종목 Watch & 정기 리포트

### 1.1 목적
사용자가 등록한 종목들에 대해 **자동으로 심층 분석**을 수행하고, 정해진 시간에 **디스코드 리포트**로 전달합니다.

### 1.2 리포트 구성
| 항목 | 설명 | 데이터 소스 |
|------|------|-------------|
| 현재가/등락률 | 당일 가격 변동 | KIS API `get_current_price` |
| 변동성 돌파 신호 | 목표가 대비 현재가 위치 | `VolatilityBreakoutStrategy` |
| 수급 동향 | 외국인/기관 3일 순매수 합계 | `SupplyAnalyzer` |
| 뉴스 키워드 | 최근 24시간 관련 뉴스 요약 | `NewsScraper` |
| 종합 의견 | BUY / HOLD / WAIT 판정 | 위 데이터 종합 |

### 1.3 발송 스케줄
- **프리마켓 리포트**: 08:30 (장 시작 전 브리핑)
- **장 마감 리포트**: 15:40 (오늘의 결과 요약)
- **위클리 리포트**: 일요일 20:00 (차주 전망)

### 1.4 관심 종목 관리
- `config/watchlist.yaml`에서 종목 코드 관리
- 대시보드(Streamlit)에서도 추가/삭제 가능하도록 연동

### 1.5 구현 파일
| 파일 | 역할 |
|------|------|
| `core/report_generator.py` | 리포트 데이터 수집 및 마크다운 생성 |
| `run_report_bot.py` | 스케줄 기반 리포트 발송 (schedule 라이브러리) |
| `config/watchlist.yaml` | 관심 종목/리포트 설정 |

## 2. MacroAnalyzer — 글로벌 거시 지표 분석 시스템

### 2.1 목적
전날 미국장 마감 결과, 환율, 채권, 원자재 등 **다양한 거시 지표를 종합 분석**하여 오늘 국내 시장의 방향성을 예측하는 프리마켓 인텔리전스를 제공합니다.

### 2.2 MacroAnalyzer 모듈 구조
```
core/macro_analyzer.py
├── class MacroAnalyzer
│   ├── get_us_indices()          # 미 3대 지수
│   ├── get_fx_rates()            # 환율
│   ├── get_bond_yields()         # 채권 금리 (Phase 2)
│   ├── get_commodities()         # 원자재 (Phase 2)
│   ├── get_crypto()              # 가상자산 (Phase 3)
│   ├── get_fear_greed_index()    # 공포탐욕지수 (Phase 3)
│   ├── get_korea_indicators()    # 한국 고유 지표 (Phase 4)
│   ├── compute_market_score()    # 종합 점수 계산
│   └── generate_comment()        # 자연어 코멘트 생성
```

### 2.3 Phase 1: 핵심 지표 (MVP)

즉시 구현할 최소 필수 지표입니다.

| 지표 | yfinance 심볼 | 한국 시장 영향 |
|------|---------------|----------------|
| 나스닥 종합 | `^IXIC` | 기술주 → 코스닥 직결 |
| S&P 500 | `^GSPC` | 미국 전체 → 코스피 연동 |
| 다우존스 | `^DJI` | 대형 가치주 → 코스피 대형주 |
| 원/달러 환율 | `USDKRW=X` | 환율↑ → 외국인 매도 압력 |
| VIX (공포지수) | `^VIX` | VIX↑ → 글로벌 리스크 확대 |

**분석 로직:**
```python
def compute_market_score(data):
    score = 0
    # 미 3대 지수 평균 등락률 (가중치 40%)
    us_avg = mean([nasdaq_chg, sp500_chg, dow_chg])
    score += us_avg * 0.4
    
    # 환율 변동 (가중치 30%) — 환율 상승은 부정적
    score -= fx_chg * 0.3
    
    # VIX 수준 (가중치 30%) — VIX 상승은 부정적 
    score -= vix_chg * 0.3
    
    # 결과: -100 ~ +100
    return normalize(score)
```

**코멘트 자동 생성:**
| 점수 범위 | 시장 코멘트 | 매매 전략 |
|-----------|-------------|-----------|
| +60 이상 | 🟢 **적극 매수** | 공격적 진입, K값 상향 |
| +20 ~ +60 | 🔵 **매수 우위** | 일반 전략 유지 |
| -20 ~ +20 | ⚪ **관망/중립** | 소극적 대응, 현금 비중 유지 |
| -60 ~ -20 | 🟡 **보수적** | 방어적, 손절 기준 강화 |
| -60 이하 | 🔴 **위험/방어** | 매수 중단, 보유 종목 점검 |

### 2.4 Phase 2: 채권 & 원자재

금리와 원자재 가격은 섹터별 영향이 다릅니다.

#### 채권 금리
| 지표 | yfinance 심볼 | 의미 |
|------|---------------|------|
| 미 2년물 국채 | `^IRX` 또는 `2YY=F` | 단기 금리 전망 (Fed 정책) |
| 미 10년물 국채 | `^TNX` | 장기 금리 → 성장주 부담 |
| 장단기 금리차 | `10Y - 2Y` (계산) | 역전 시 경기침체 경고 |

> **활용**: 10년물 금리 급등 → 성장주(코스닥) 약세 예상, 가치주 전환 고려

#### 원자재
| 지표 | yfinance 심볼 | 관련 국내 섹터 |
|------|---------------|----------------|
| WTI 원유 | `CL=F` | 정유/화학 (SK이노, S-Oil) |
| 금 | `GC=F` | 안전자산 선호도, 인플레이션 지표 |
| 구리 | `HG=F` | 경기 선행지표 ("Dr. Copper") |
| 천연가스 | `NG=F` | 에너지 섹터 |

> **이 지표들을 리포트의 "섹터별 전망" 섹션에 연결합니다.**

## 3. 해외 커뮤니티 감성 분석 (Reddit)

### 3.1 목적
Reddit의 투자 커뮤니티에서 **시장 심리(Sentiment)**를 수집하여, 글로벌 투자자들의 관점을 선행 지표로 활용합니다.

### 3.2 대상 서브레딧
| 서브레딧 | 특성 |
|----------|------|
| `r/wallstreetbets` | 고변동성 단타, 밈주식, 급등 예측 |
| `r/stocks` | 일반 주식 토론, 중장기 |
| `r/investing` | 가치투자, 매크로 |

### 3.3 분석 방법
1. **트렌딩 티커 추출**: `$NVDA`, `$TSLA` 등 언급 빈도 카운팅
2. **감성 점수 계산**:
   - Bullish 키워드: `buy`, `moon`, `calls`, `long`, `green`, `bull`
   - Bearish 키워드: `sell`, `crash`, `puts`, `short`, `red`, `bear`
   - Score = `(Bull - Bear) / (Bull + Bear)` → -1.0 ~ 1.0
3. **Hot Ticker 알림**: 특정 종목 언급이 급증하면 디스코드 알림

### 3.4 데이터 수집
- **라이브러리**: `praw` (Python Reddit API Wrapper)
- **Reddit API 키 필요**: https://www.reddit.com/prefs/apps 에서 발급
- `config/secrets.yaml`에 Reddit 자격증명 추가

### 3.5 구현 파일
| 파일 | 역할 |
|------|------|
| `core/reddit_analyzer.py` | Reddit 데이터 수집 및 감성 분석 |
| `config/secrets.yaml` | Reddit API 자격증명 추가 |
